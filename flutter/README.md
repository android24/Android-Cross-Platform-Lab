# flutter

Flutter 实验工程目录。

本目录用于实现同一贯穿项目的 Flutter 版本，并重点验证 Widget 体系、布局、导航、状态管理、数据库、Platform Channel、Plugin、Flutter Module 和性能优化。

## 目标范围

- Dart 与 Flutter 项目结构
- 目录、详情、搜索、收藏、历史和设置页面
- 网络、数据库、缓存、主题和多语言
- Platform Channel
- Flutter Plugin
- Flutter Module 接入 Android 原生壳
- Flutter DevTools 性能分析
