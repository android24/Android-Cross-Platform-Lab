# react-native

React Native 实验工程目录。

本目录用于实现同一贯穿项目的 React Native 版本，并重点验证 TypeScript、RN 组件、导航、状态管理、缓存、Native Module、Native Component 和性能调试。

## 目标范围

- TypeScript 与 React Native 项目结构
- 目录、详情、搜索、收藏、历史和设置页面
- 网络请求、缓存、主题和多语言
- Kotlin Native Module
- Android Native Component
- 原生与 RN 混合接入
- 启动、内存和帧率采集
